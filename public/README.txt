Public Assets Notes
===================

This starter intentionally includes no image files.

When ready, place production logo assets in this folder, for example:
- /public/logo-primary.png
- /public/logo-mark.png
- /public/logo-light.png

After adding logo files, update the text-logo placeholders in:
- /components/Header.jsx
- /components/Footer.jsx

Do not commit temporary draft assets that are not optimized for web use.
